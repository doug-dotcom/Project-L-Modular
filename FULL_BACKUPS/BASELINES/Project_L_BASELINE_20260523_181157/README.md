# Project L

Project L is a modular AI assistant architecture built for:

- grounded AI interaction
- memory persistence
- orchestration
- emotional continuity
- modular expansion
- Railway deployment

## Features

- FastAPI backend
- OpenAI integration
- Memory engine
- JSON persistence
- UI chat interface
- Structured logging
- Startup validation
- Railway-ready deployment

## Launch

uvicorn api.server:app --host 0.0.0.0 --port 8000
